# Security Policy

## Supported Versions

Currently supported versions:

| Version      | Supported |
| ------------ | --------- |
| indev builds | yes       |

## Reporting a Vulnerability

If you find a security issue in WasteRam, please report it by opening a GitHub issue.

Please include:

* description of the problem
* steps to reproduce
* screenshots if possible

Thank you for helping make WasteRam safer.
