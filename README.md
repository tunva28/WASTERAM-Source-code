# WASTERAM-Source-code
Source code of WASTERAM

# What is WasteRam?

WasteRam is a completely useless terminal simulator where every command wastes a little more RAM and a little more of your life.

Type random commands. Open suspicious videos.

Congratulations — you are now a certified WASTERAM™.

still in INDEV

# Features

- Command-based gameplay
- Achievement system
- Terminal animations
- ASCII art
- Meme video commands
- Fake OS simulation (SolnaMOS)
- Sound effects
- Hidden easter eggs

Safe Python program compiled with PyInstaller.
No internet access except opening YouTube links.
